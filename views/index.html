Responsive Design

Es los que hace que una pagina web se vea correctamente independientemente del dispositivo desde el que se visualiza.

Breakpoints

Es un momento donde la pagina web cambia. Deben servir como referencia para crear nuevos breakpoints según lo necesite.

Resolución

Se tiene en cuenta el ancho de pixeles del dispositivo.

Básicos

Son los pixeles de los principales dispositivos mas genéricos, por ejemplo: Celular, Tablet, Laptop(computadora).

Completos

Se amplia rango de dispositivos a utilizar, como versiones mas pequeñas o grandes de un determinado dispositivo.

Personalizado

Es cuando el diseño | maquetación de una pagina web se rompe en un pixel concreto.

Dispositivo

Se tiene en cuenta el comportamiento del propio dispositivo.

  • Orientación
  • Modo Oscuro
  • Pantalla de impresión (No se usa)

Media Queries

Declaración de CSS que define mediante breakpoints en qué pixeles se cambiara las propiedades de un elemento.

Sintaxis:

  • Keyword: @media
  • Media Type: screen
  • Operador: and|or|not
  • Breakpoint | Media Feature: (property : value)

Ejemplos

@media screen and|or|not (max-width:480px) {

}

@media (min-width:480px) and (max-width:768px) {

}

@media (orientation:landscape) and (max-width:480px) {

}

@media (prefers-color-scheme:dark) {

}
